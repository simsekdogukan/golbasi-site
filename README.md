# Gölbaşı Digital Menu

A premium, mobile-first digital menu application for Gölbaşı Restaurant.

## Features
- **Premium Design**: Glassmorphism UI with gold/black theme.
- **Multi-language**: Turkish (TR), English (EN), Arabic (AR).
- **Dynamic Data**: Powered by JSON data converted from Excel.
- **Responsive**: Fully optimized for mobile devices.

## Setup
1.  Clone the repository.
2.  Run `npm install` to install dependencies.
3.  Run `npx http-server` to start locally.

## Deployment
This project is ready for Vercel.
1.  Import this repository in Vercel.
2.  Deploy!
